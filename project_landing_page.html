IHG Nudge Calendar Toolkit

This internal tool helps generate 365-day training calendars using pre-written nudges and exports them to an iCalendar format.

📁 Files

  • extract_nudges.py – Converts .txt documents into structured Excel files
  • generate_schedule.py – Generates 365-day calendar with rotating categories
  • generate_ics.py – Exports final .xlsx calendar to .ics format
  • requirements.txt – Lists required dependencies

📌 Usage

Run these in your terminal after installing dependencies:

pip install -r requirements.txt
python extract_nudges.py
python generate_schedule.py
python generate_ics.py
